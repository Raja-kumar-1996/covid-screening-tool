A simple yet powerful **COVID-19 Symptom Checker** built using Python.  
This project evaluates risk levels based on user symptoms and temperature input.

---

## 🚀 Features

- 🌡️ Temperature-based risk detection  
- 🤒 Symptom-based scoring system  
- 🔁 Multi-user support (loop system)  
- 💾 Saves results to a file (`results.txt`)  
- 🧠 Beginner-friendly logic  
- ⚡ Fast and lightweight CLI app
